Jupyter-notebooks Tutorials

To run these examples you will need the nightly Conda build of CDAT installed with Jupyter.

If you are unfamiliar with Jupyter notebooks, many resources exist on the web including the Jupyter Project's home page or Jupyter's Documentation page.

Other learning resources include:

  • Dataquest's Jupyter Notebook Tutorial
  • Read the Docs' Jupyter/IPython Notebook Quick Start Guide
  • DataCamp's Jupyter Notebook Tutorial: The Definitive Guide
